# Documentation

This directory contains the extended documentation for GhostCred.

## Overview
While the main README provides a quick start, this directory contains deep-dive documentation on:
- **Architecture**: How GhostCred works under the hood, how the plugin system (pluggy) operates, and how lineage tracking computes blast radius.
- **Enterprise Features**: Explanations of the Compliance Mapping engine (SOC 2, ISO 27001), Webhook/SIEM integration patterns, and Time-To-Remediation (TTR) SLO configuration.
- **Deployment**: Best practices for running GhostCred in CI pipelines (GitHub Actions, GitLab CI) and as an active background watcher on developer workstations.
